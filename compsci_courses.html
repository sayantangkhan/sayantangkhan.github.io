Introductory Data Structures and Algorithms (Semester 1)

After a basic introduction to the C programming language, the course moved on to elementary data structures like, linked lists, trees, graphs, etc. It then focused on various algorithms using these data structures like Dijkstra's algorithm for minimal path, Prim's and Kruskal's algorithms for minimal spanning tree.

Cryptography (Semester 4)

This course focused on proofs of security of various protocols, most of the proofs being reductions to known hard problems. We covered encryption protocols, message authentication protocols, and signing protocols, both in the symmetric and asymmetric key setting.

Randomized Algorithms (Semester 4) (Not credited)

This course focused on looking at various algorithms that used randomness in the course of their execution, and how that affected their running complexity. We also developed many approximation methods in the course of this analysis, like Lovász local lemma, Markov chains, etc.

Automata Theory and Computability (Semester 5)

This course focused on automatons, i.e. finite automatons, pushdown automatons, and Turing machines, and how they were useful in solving decision problems in formal logic. The capstone of the course was a proof of Gödel's First Incompleteness Theorem using computable reductions to the negation of the Halting Problem which is not recursively enumerable.
