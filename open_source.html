i3blocks Blocklets

i3wm is a tiling window manager for GNU/Linux and BSD based operating systems. Blocklets are pluggable modules that display various pieces of information on a bar at the top or bottom of the screen. I have written a few blocklets of my own as well, and sent a pull request back to the main project so that they can be integrated in the default configuration. Battery bar is a forked blocklet, which shows the remaining battery on the bar. display_song.py is a blocklet that displays the currently playing song, as well as artist and album details, and interactively allows song controls without having to open the media player.

Pokémon GO Assistant

PokemonGoAssistant was a web app I wrote over the course of 24 hours that used the reverse engineered API of Pokémon GO to let me scout for Pokémon in nearby areas, so that I could look for rare Pokémon without having my phone out and running all the time.
